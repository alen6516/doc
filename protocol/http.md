# Structure

## Field description
* Start-line
    * request method: GET, POST, PUT, DELETE, HEAD

* Host
    * hostname, which is mandatory in HTTP/1.1

* User-Agent
    * client's information

* Content-Type
* Content-length
    * len of the content

* Accept-Language
* Accept-Encoding
* Accept-Charset
* Keep-Alive
* Connection
